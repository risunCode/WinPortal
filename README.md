## 🔧 Bypass OOBE New 24H2+

### 🚫 Bypass Windows 11 OOBE (24H2+)
When stuck at the Out-of-Box Experience screen:
1. Press `Shift + F10` to open Command Prompt
2. Run:
 ```
start ms-cxh:localonly
 ```
